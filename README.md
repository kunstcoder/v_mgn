# MeshGraphNet Demo

이 저장소는 **MeshGraphNet 입력/학습/출력 시각화**를 위한 최소 실행 가능한 정적 웹 데모 구조를 제공합니다.

## 구성

- `index.html`: 데모 화면의 기본 구조
- `styles.css`: 레이아웃 및 UI 스타일
- `app.js`: 샘플 데이터 로딩 및 텍스트 렌더링 로직
- `data/`: 확장 가능한 샘플 JSON 데이터

## 실행 방법

별도 빌드 없이 `index.html`을 브라우저에서 열어 UI 구조를 확인할 수 있습니다.

> 참고: 브라우저 보안 정책(CORS)으로 인해 `fetch` 기반 파일 로딩이 제한될 수 있으므로, 필요 시 간단한 로컬 서버를 사용하세요.
